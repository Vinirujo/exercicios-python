# Exercícios de Python

Coleção de pequenos programas desenvolvidos durante meus estudos de lógica de programação e Python. Cada arquivo é independente e focado em um conceito específico.

## Scripts

| Arquivo | Descrição |
|---|---|
| `adivinhacao.py` | Jogo de adivinhação de números de 1 a 100 |
| `par_ou_impar.py` | Verifica se um número é par ou ímpar |
| `numeros_invertidos.py` | Inverte a ordem dos dígitos de um número de 4 dígitos |
| `calculo_nota_if_else.py` | Calcula a média de duas notas com condicionais if/else |
| `calculo_media_interativo.py` | Calcula a média de 4 notas informadas pelo usuário |
| `calculo_media_fixo.py` | Cálculo de média usando função com valores fixos |
| `calculadora_com_funcao.py` | Calculadora simples (+, -, *, /) usando função |
| `menu_calculadora.py` | Calculadora com menu interativo em loop |
| `converter_texto_print.py` | Exercício de formatação de saída com f-strings |
| `pedra_papel_tesoura.py` | Jogo de Pedra, Papel e Tesoura contra o computador, com placar |
| `gerenciador_numeros.py` | Gerencia uma lista de números: adicionar, remover e listar |

## Tecnologias
- Python 3

## Como executar
Cada arquivo pode ser executado individualmente:
```bash
python nome_do_arquivo.py
```

## O que aprendi
Prática de conceitos fundamentais: condicionais, laços de repetição, funções, listas, manipulação de strings e menus interativos.
